# P2P Encrypted Chat

## Overview
A secure, peer-to-peer encrypted chat application that allows direct communication and file sharing without intermediary servers.

## Features
- End-to-end encryption
- WebRTC peer connection
- File sharing
- Works behind NAT networks
- No central server required

## How to Use
1. Open HTML file in modern browser
2. Create token (Host)
3. Client connects using token
4. Enter SAME passphrase
5. Start chatting!

## Requirements
- Modern web browser (Chrome, Firefox, Edge)
- WebRTC support

## Security
- AES-GCM encryption
- Passphrase-based key derivation
- Direct peer connections

## Disclaimer
Use at your own risk. Not recommended for sensitive communications.
